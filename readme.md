<h1>Chatting Robot </h1>

## 1、写这个项目的目的

主要的练习对第三方平台的使用，因为作为学生，之前我们在学校写的项目都是从0开始，所有东西都是自己实现，但是如果进入公司，不免要使用别人已经写好了的东西，所以这个项目主要是用来学习第三方平台的使用；还有一个原因就是感觉这个项目比较有意思。
